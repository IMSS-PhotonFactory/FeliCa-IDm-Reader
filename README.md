# FeliCa-IDm-Reader

## 概要
Windows上でFeliCaリーダーを用いてFeliCa ID（IDm）を読み取るアプリケーションです。

## 動作確認環境
Windows 11
リーダー：SONY RC-S380
※コントロールパネルから「NFCポート/パソリ」で下記を調整（スマホなどマルチNFC対策）
* PC/SC有効
* 検出対象ICカードをNFC-Fに限定

## 依存関係
* PCSC 6.1.3
* PCSC.Iso7816 6.1.3
